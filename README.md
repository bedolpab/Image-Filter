How to use:

1.) Upload 3 Images to the same folder in which the .py files are located

2.) Rename each of those files (NOTE: NAMES CAN BE CHANGED IN main.py):

a: subjectIMG.jpg

b: subjectIMG0.jpg

c: subjectIMG2.jpg

  
3.) Run program, and follow the simple steps prompted
